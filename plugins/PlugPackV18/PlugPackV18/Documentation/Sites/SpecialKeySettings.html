Special Key Settings
 
This list contains all information about the not so often used key settings.
 
Mainscreen Key Settings:
Option (Alt) + Closest Targ Nearest friendly/escort ship (green) or nearest standard ship (yellow)
Option (Alt) + Target Select Next ship (includes friendly/escort and enemy)
Option (Alt) + Select Secondary Select secondary (backwards)
Option (Alt) + Communicate If a planet/station AND a ship is selected, you can speak to the planet/station
Map Key Settings:
C Center view to current position
Tab Automatic jumpcourse to a system near you (clockwise)
+ And - Enlarges and reduces the map
Player Info Key Settings:
Tab Next information (next page)
Commodity Key Settings:
Option (Alt) + Buy Enter quantity of junk to buy
Outfit Key Settings:
Option (Alt) + Buy Enter quantity of items to buy
Gamble Key Settings:
Option (Alt) + Klick Enter quantity of credits to bet
